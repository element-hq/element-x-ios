## Описания приложения

### Русский (подтверждено ✅)

**Полное описание:**
```
UCMeet.Chat — защищённый мессенджер для безопасного общения вашей организации.

Общайтесь с коллегами, используя сквозное шифрование сообщений, голосовые и видеозвонки, обмен файлами и медиа.

ВОЗМОЖНОСТИ:
• Сквозное шифрование всех сообщений (протокол Matrix)
• Голосовые и видеозвонки
• Обмен фото, видео, файлами и геолокацией
• Групповые чаты и пространства (Spaces)
• Синхронизация сообщений между устройствами
• Блокировка приложения по Face ID / Touch ID

БЕЗОПАСНОСТЬ:
Все сообщения защищены сквозным шифрованием по протоколу Matrix (Olm/Megolm). Только вы и ваши собеседники можете прочитать переписку.

ТРЕБОВАНИЯ:
Для использования приложения необходима учётная запись UCMeet. Обратитесь к администратору вашей организации для получения доступа.

Построено на открытом протоколе Matrix.
```

**Рекламный текст:**
```
Защищённый корпоративный мессенджер со сквозным шифрованием. Звонки, чаты и файлы — всё в одном приложении.
```

**Ключевые слова:**
```
мессенджер,чат,matrix,шифрование,безопасность,звонки,корпоративный,приватный,UCMeet
```

### Английский (подтверждено ✅)

**Полное описание:**
```
UCMeet.Chat — secure messenger for your organization's private communications.

Communicate with colleagues using end-to-end encrypted messaging, voice and video calls, and file sharing.

FEATURES:
• End-to-end encrypted messaging (Matrix protocol)
• Voice and video calls
• Photo, video, file, and location sharing
• Group conversations and Spaces
• Cross-device message sync
• Biometric app lock (Face ID / Touch ID)

SECURITY:
All messages are protected with end-to-end encryption using the Matrix protocol (Olm/Megolm). Only you and your intended recipients can read your messages.

REQUIREMENTS:
A UCMeet account is required. Contact your organization's administrator for access.

Built on the open Matrix communication protocol.
```

**Рекламный текст:**
```
Secure enterprise messenger with end-to-end encryption. Calls, chats, and files — all in one app.
```

```
messenger,chat,matrix,encryption,secure,calls,enterprise,private,UCMeet
```
