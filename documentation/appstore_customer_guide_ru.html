Инструкция для заказчика
Цель документа: собрать от заказчика всю информацию, необходимую для публикации приложения UCMeet.Chat в Apple App Store. Все технические настройки приложения выполнены. После предоставления данных по пунктам ниже публикация займёт 1–2 рабочих дня, плюс проверка Apple (обычно 1–3 дня).

Шаг 7. Юридические вопросы

7.1. Лицензия AGPL v3

Приложение основано на открытом коде Element X iOS (лицензия AGPL v3). Для публикации в App Store необходимо письменное подтверждение, что лицензия AGPL v3 покрывает этот форк.

БЛОКИРУЮЩИЙ ВОПРОС: Без письменного подтверждения лицензии AGPL v3 мы не можем отправить приложение на ревью Apple. Предоставьте подтверждение в виде email или документа.

7.2. Декларация шифрования

Приложение использует сквозное шифрование (E2EE) через Matrix SDK. По правилам BIS (Bureau of Industry and Security, США) требуется ежегодная самоклассификация.

ПараметрЗначение
Использует шифрованиеДа
ТипСквозное (Olm/Megolm через Matrix Rust SDK)
Классификация ECCN5D002
ИсключениеМассовое распространение (EAR §740.17(b)(1))
Настройка в приложенииITSAppUsesNonExemptEncryption = true (готово)

Для подачи самоклассификации потребуются данные вашего юридического лица:

Тип авторизации: Mass Market (EAR §740.17(b)(1))
Нужно ваше действие: Укажите официальное название юридического лица для подачи самоклассификации.
